# Game Arena 🎮

A collection of console-based games built using Java.

This project was created to improve problem-solving, game logic, file handling, and overall programming skills through interactive mini games.

---

## Games Included

### Connect Four
A two-player strategy game where players try to connect four pieces in a row.

Features:
- Two-player gameplay
- Win detection system
- Input validation
- Game board visualization

---

### Word Scramble
A word guessing game where players guess the correct word from scrambled letters.

Features:
- Random word generation
- Multiple categories
- Score tracking
- File handling using text files
